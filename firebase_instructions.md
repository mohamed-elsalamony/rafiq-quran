# دليل إعداد ونشر خادم المساعد الذكي على Firebase Cloud Functions

تم تصميم الخادم للعمل مباشرة كخدمة سحابية بدون خادم (Serverless) على **Firebase Cloud Functions** في بيئة Node.js 18. يوفر هذا حلاً مجانياً بالكامل (ضمن الحدود المجانية السخية لفايربيز) مع حماية كاملة لمفتاح Gemini API الخاص بك.

---

## المتطلبات الأساسية
1. تثبيت [Node.js](https://nodejs.org/) (إصدار 18 أو أعلى).
2. حساب على [Firebase Console](https://console.firebase.google.com/) وإنشاء مشروع جديد.

---

## خطوات الإعداد والنشر

### الخطوة 1: تثبيت أدوات Firebase (Firebase CLI)
افتح الطرفية (Terminal) في جهازك وقم بتشغيل الأمر التالي لتثبيت أدوات فايربيز عالمياً:
```bash
npm install -g firebase-tools
```

### الخطوة 2: تسجيل الدخول إلى حساب Firebase
قم بتشغيل الأمر التالي لربط الـ CLI بحساب جوجل الخاص بـ Firebase:
```bash
firebase login
```
سيفتح المتصفح لتسجيل الدخول والموافقة على الصلاحيات.

### الخطوة 3: ربط المجلد بمشروعك في Firebase
في المجلد الرئيسي للمشروع (حيث يوجد ملف `firebase.json` و مجلد `functions`)، قم بربط المشروع بالتشغيل:
```bash
firebase use --add
```
سيظهر لك قائمة بمشاريع Firebase الخاصة بك، اختر المشروع الذي أنشأته للمصحف، وقم بتسمية الاسم المستعار (Alias) بـ `default`.

### الخطوة 4: إعداد مفتاح الـ API للإنتاج
افتح مجلد `functions` وقم بفتح ملف `.env` الذي قمنا بإنشائه، وأدخل مفتاح Gemini API الخاص بك فيه:
```env
GEMINI_API_KEY=AIzaSy...
GEMINI_MODEL=gemini-2.5-flash
```
*ملاحظة: في الجيل الثاني من Firebase Functions (v2)، يتم رفع ملف `.env` تلقائياً واستخدامه لتأمين وتغذية خوادم فايربيز بالمتغيرات البيئية بأمان.*

### الخطوة 5: تثبيت حزم الخادم ونشره (Deploy)
افتح الطرفية في مجلد `functions` وقم بتثبيت الحزم المطلوبة:
```bash
cd functions
npm install
```
ثم عد للمجلد الرئيسي للمشروع أو من مجلد `functions` وشغّل أمر النشر:
```bash
firebase deploy --only functions
```

## بعد النشر بنجاح: ربط التطبيق

عند اكتمال النشر، ستقوم Firebase بطباعة رابط النهاية (Endpoint URL) في الطرفية. سيبدو الرابط كالتالي:
`https://api-<عشوائي>.run.app` (أو يبدأ بـ `cloudfunctions.net`).

1. انسخ هذا الرابط بالكامل.
2. افتح تطبيق **رفيق القرآن** على هاتفك.
3. اذهب إلى **شاشة الإعدادات** -> قسم **إدارة حالة المساعد (AI Backend)**.
4. الصق الرابط في حقل **عنوان سيرفر الـ Backend** (تأكد من إبقاء الرابط بدون شرطة مائلة مضافة بنهايته).
5. اضغط على **حفظ وتفعيل**.
6. سيقوم التطبيق بفحص الرابط وتغيير الحالة فوراً إلى **متصل بالمساعد الذكي (Online)** باللون الأخضر.
